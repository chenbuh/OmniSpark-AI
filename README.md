# OmniSpark AI — 统一生图与视频创作中台

全栈 AI 创作平台，在一个账号内完成文生图、图生视频、视频编辑、工作流编排、团队协作等任务。

## 技术栈

| 层级 | 技术 |
|---|---|
| **前端** | Vue 3 + TypeScript + Vite 8 + Naive UI + Pinia + Axios + Vue Router |
| **后端** | Java 17 + Spring Boot 3.0.5 + MyBatis-Plus 3.5.15 + MySQL + Redis |
| **鉴权** | Sa-Token 1.45.0（Redis 存储 Session） |
| **安全** | BCrypt 密码哈希 + RSA-OAEP 密码传输加密 + HMAC 请求签名防重放 + 项目级越权防护 + 接口限流 + 自研多形态验证码 |
| **实时通信** | WebSocket (STOMP) 通知推送 |
| **PWA** | vite-plugin-pwa 离线缓存 |
| **AI API** | OpenAI 兼容接口 / 自定义 HTTP 服务商 |
| **MCP** | Puppeteer / Browser-use / Fetch / Filesystem |

## 功能清单

### 🎨 图像生成
- **文生图** — 输入 Prompt，AI 生成图像
- **图生图** — 上传参考图生成变体
- **局部重绘 (Inpainting)** — Canvas 画笔绘制遮罩，AI 只重绘标记区域
- **多参考图** — 最多 16 张参考图，支持拖拽上传
- **批量生成** — 一次最多 8 张，缩略图网格切换
- **流式进度** — 后端实时进度推送（10%→20%→40%→80%→100%），2 秒轮询
- **参数对比** — 生成结果中显示实际使用的参数（尺寸、CFG、Steps、模型等）

### 📋 任务管理

### 🔒 安全
- **密码存储** — BCrypt（cost=12，自带随机盐）哈希；登录时平滑升级历史 SHA-256 与低 cost 哈希
- **密码策略** — 最小长度 8 位、常见弱密码黑名单、禁止与账号相同
- **传输加密** — 前端用 RSA-OAEP(SHA-256) 加密密码后提交，后端私钥解密；默认关闭明文降级通道
- **请求签名** — 认证接口与管理员用户写操作校验 `X-Timestamp / X-Nonce / X-Sign`，Redis nonce 去重防重放
- **越权防护** — 项目级 `ProjectAccessGuard` 统一校验，生成/工作流/字幕等资源严格按归属过滤
- **接口限流** — 注解式 `@RateLimit` + Redis 固定窗口，支持 IP / 用户 / 用户+接口三维度；已挂登录、注册、验证码、生成接口（生成为 10 次/分钟 + 200 次/天双闸）
- **自研验证码** — 登录/注册前置多形态滑块验证码（旋转校正 / 顺序点击 / 轨迹拖动随机切换），图料后端纯代码生成、答案不下发，叠加"像人"行为风控与 IP 失败计数
- **凭证管理** — 数据库账号、签名 secret 等通过环境变量注入，不写入仓库

> 防爬取 / 防接口滥用的分层加固方案（限流、生成频率控制、多形态验证码、分页上限、安全响应头等）与"浏览器无法隐藏接口"的认知澄清，见 `docs/接口防爬取与防滥用加固方案.md`。

## 快速启动

### 前置条件
- JDK 17+
- MySQL 8.0+
- Redis
- Node.js 20+

### 后端

数据库账号密码通过环境变量注入，不写入仓库。`application.yml` 中已配置占位：
`username: ${DB_USERNAME:root}`、`password: ${DB_PASSWORD:}`。
可参考根目录 `.env.example` 模板（复制为 `.env` 填入真实值，`.env` 已被 gitignore 排除）。

启动前先设置环境变量（任选一种方式）：

```bash
# macOS / Linux
export DB_USERNAME=root
export DB_PASSWORD=你的数据库密码

# Windows PowerShell
$env:DB_USERNAME="root"; $env:DB_PASSWORD="你的数据库密码"
```

IntelliJ IDEA 用户：在 `Run/Debug Configurations → Environment variables` 里添加
`DB_USERNAME` 与 `DB_PASSWORD`，无需修改 `application.yml`。

```bash
cd backend
mvn spring-boot:run
# 启动后自动建表 + 初始化管理员账号 admin / admin123
```

> 默认连接 `jdbc:mysql://127.0.0.1:3306/aihub`，Redis 连接 `127.0.0.1:6379`。
> 如需调整主机/端口/库名，改 `application.yml` 对应项即可。

### 前端
```bash
npm install
npm run dev
```
